## 递归
```
    public static void sort(int a[], int step) {
        for (int i = 1; i <= 9; i++) {
            a[step] = i;
            if (step == 8) {
                rule(a);
            }
            if (circulate(a, step - 1))
                break;
            if (step < 8) {
                sort(a, (step + 1));
                continue;
            }
        }
    }
```
